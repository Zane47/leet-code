## 翻转链表
```java
// 反转以 a 为头结点的链表
public class Test {
    ListNode reverse(ListNode a) {
    ListNode pre, cur, nxt;
    pre = null; cur = a; nxt = a;
    while (cur != null) {
        nxt = cur.next;
        // 逐个结点反转
        cur.next = pre;
        // 更新指针位置
        pre = cur;
        cur = nxt;
        }
    // 返回反转后的头结点
    return pre;
    }
}

```
